AutoCAD笔记

发表于 2019-03-31

README

本文记录在学习使用AutoCAD过程中的操作心得

初识CAD

CAD界面及编辑

（1）界面分为快速访问工具栏、开始工具、工具栏、绘图区、命令栏、辅助状态栏。
（2）工具栏上右键可以对分区进行显示和隐藏设置。选项卡右边小三角，可让工具栏最小化。
（3）鼠标在工具图标上停留，可以弹出相应帮助。
（4）点击辅助状态栏里的齿轮图标可以切换工作空间，有恢复所有界面设置的作用。

CAD基础操作及理论

CAD新建打开及保存

（1）新建文件有多种方式，快速访问工具栏里新建、开始里新建、快捷键ctrl+N，一般我们选择无样板打开公制。
（2）界面最上方显示的是当前文件的文件名，标准CAD文件后缀是DWG。
（3）保存可以点击快速访问工具栏里的保存，或用快捷键ctrl+s。
（4）保存可以自定义路径、名称以及版本。

鼠标基础操作

（1）鼠标左键是点击选中。
（2）鼠标中键前滚是放大，后滚是缩小，它是以鼠标为中心的。按住不放拖动鼠标是平移，双击中键是显示全部。
（3）鼠标右键是菜单键。

CAD选择技巧

（1）点选：左键点击；shift+左键点击是取消选择；ESC是退出。
（2）矩形框选：左键点击对角点，由左往右是框选住全部才能选中对象，由右往左是框选住部分即可选中对象。
（3）全选：快捷键ctrl+A；当执行某个命令，让选择对象的时候，也可以直接输入all，然后空格是全选。
（4）栏选：执行命令时，提示选择对象，输入F，画折线，所有与折线相交的对象被选中。

CAD直线工具

（1）直线工具创建的对象本质是线段，而不是无线延伸的直线。
（2）注意命令行提示，可以空格或者回车键来确认退出，而ESC是直接退出。
（3）快捷键L；直接在界面里输入；养成使用快捷键的习惯。
（4）动态辅助快捷键F12，tab键切换角度和长度，角度和长度均为正值，注意角度需要用回车键确认的。
（5）注意辅助状态的图标。回撤可以使用快捷键ctrl+z。

CAD相对坐标

（1）绝对坐标：指该点相对于原点的X（水平）/Y（竖直）的坐标值。
（2）相对坐标：指该点相对于上一点的X（水平）/Y（竖直）的坐标值，以(X,Y)的形式输入，注意中间的逗号是英文半角符号。
（3）x值和y值的形式：X值左负右正，Y值上正下负。
（3）与动态输入的区别：动态输入是输入长度和角度，用tab切换；相对坐标输入是水平相对距离和竖直相对距离，用英文逗号切换。

CAD极坐标

（1）极坐标：带有角度的坐标值，以（距离<角度）形式输入，角度指的是直线与水平向右的夹角。
（2）角度值逆时针为正，顺时针为负。
（3）与动态输入区别，极坐标的角度是分正负的。

其他

（1）空格可以重复上一命令。
（2）捕捉设置：可在捕捉图标上右键设置捕捉各种类型。

CAD基础工具

二维构图：圆

（1）快捷键：C
（2）默认是指定圆心半径作圆，命令行括号内的值是当前的默认值。
（3）”相切相切相切作圆”命令行里没有这个功能，只能点击图标启动。
（4）只有进入某个命令的时候才能捕捉圆心。

二维构图：圆弧

（1）快捷键：ARC
（2）圆弧形成的方向默认是逆时针，可以按ctrl来切换方向。指的是圆弧两个端点与圆心连线所成夹角。
（3）弦长指的是圆弧两端点之间的距离，最大不能超过圆弧的直径，否则会绘制失败。

二维构图：椭圆

（1）快捷键：EL
（2）椭圆的角度是根据起点的位置，逆时针慢慢增加的。

二维构图：矩形

（1）快捷键：rec
（2）矩形工具输入方式：(X,Y)，跟相对坐标一样，是分正负的。
（3）X值左负右正，Y值上正下负。

二维构图：多边形

（1）快捷键：pol
（2）多边形工具只能绘制正多边形，即边长及内角都相等。
（3）多边形的边数必须大于等于3

二维构图：多线

（1）快捷键：ml
（2）对正：
上：在光标下方绘制多线，参照线是上面的线。无：光标在中间绘制多线，参照线即两线的中心线。下：在光标下方绘制多线，参照线即最下面的线。。
（3）多线编辑方式在菜单里的修改——对象——多线。

二维构图：构造线及射线

（1）构造线快捷键：XL
（2）构造线是无限延伸的直线，一般用作辅助线。
（3）选择构造线要用鼠标从右到左去框选，选择部分即可选中对象。
（4）射线快捷键：ray，不是很常用，了解即可。

二维构图：多段线

（1）快捷键：PL
（2）多段线是一个整体，可以连续绘制直线和圆弧。
（3）多段线可以修改宽度。
（4）利用多段线中修改宽度的功能，在末端设定宽度为大于0，在前端设置宽度为0，可以画出箭头形状
（5）多段线利用x分解成多个对象

二维构图：样条曲线

（1）快捷键：spl
（2）一般用于剖面图的分界线。
（3）老版本需要按四次空格退出样条曲线，2014版本以后的直接可以空格退出。

二维构图：图案填充

（1）快捷键：H
（2）创建方式：A.拾取点方式，选择封闭区域内部的点。B.选择对象方式，在图案填充选项卡内点击选择按钮，顺时针或逆时针选择边界创建，不封闭的区域也可以创建，但是会有图案信息丢失。
（3）图案填充里，任何竖直的改变必须用回车进行确认
（4）无法确定闭合边界时会有红色圈提示，使用快捷键re重生成图形
（5）当填充图案无法显现时，检查图案填充比例看是否太小或太大

CAD编辑工具

移动工具

（1）快捷键：M
（2）编辑工具一般可以先选择编辑对象，然后输入快捷键激活命令。
（3）基准点是与图案保持相对距离的点，选择拾取恰当的基准点有利于精准移动

旋转工具

（1）快捷键：RO
（2）同样先选择对象，然后选择基点，即绕着哪一点进行旋转。
（3）旋转带有可以选择的复制功能
（4）旋转参照：指定基点，输入R启动参照，再次选择基点，选择对象上任意一点，再点选参照对象上的点。

修剪延伸工具

（1）修剪快捷键：TR 延伸工具快捷键：EX
（2）创建方式：
输入快捷键，直接按两次空格，即默认选择所有对象，点选修剪或延伸的对象。
（3）修剪注意：只有交点而没有突出部分的线是无法修剪的，只能手动删除。
（4）延伸注意，视口能看得到的边界才能延伸成功。
（5）突出部分线段指的是在进入本次修剪前就已经存在的那些线段。

删除及复制工具

（1）删除快捷键：E 复制快捷键：co
（2）删除时候先选中删除的对象，然后输入快捷键E，空格确定。
（3）复制工具一次性可以复制多个，加深理解下复制基点的选择，选择与对象相对位置不变的点。
（4）在命令中需要定位的时候可以利用拾取点进行精确定位。
（4）复制工具有阵列功能，可以复制多个对象，其中阵列里的布满命令较为常用。
（5）阵列的布满命令：先选择起始位置，再选择结束位置，则可以在这两个位置范围内均匀分布所复制对象。

镜像工具

（1）快捷键：MI
（2）先选择要镜像的对象，确认后选择镜像线上的两点，确认后完成。

倒圆角工具

（1）快捷键：F
（2）启动倒圆角工具之后要检查圆角默认值是否需要更改。
（3）倒角设置为0，可以起到建立直角的作用。

分解工具

（1）快捷键：X
（2）先选择对象，然后输入快捷键X，将整个对象分解为可以单独编辑的对象。
（3）一个整体的图形是无法进行更改属性等操作的，分解后才可以。
（4）对于插入的块，如果其属性是无法分解，则该工具无效。

拉伸工具

（1）快捷键：S
（2）选择拉伸的对象的时候一定要从右往左选择（框选），且选择部分对象。
（3）从左往右选择（框选）是移动功能。

缩放工具

（1）快捷键：SC
（2）与中键的区别，缩放工具改变了图形真实尺寸。
（3）选择基点后输入比例因子缩放。比例因子有多种方式输入，可以输入放大/缩小倍数，也可以输入分式，新值在前，旧值在后。
（4）自带参照功能，起到对齐的作用。

倒角工具

（1）快捷键：cha
（2）倒角距离1对应的就是第一次选择的直线，倒角距离2对应的是第二次选择的直线。
（3）以角度倒角指的是第一次选择的直线与倒角后形成的缺口线段的夹角，并没有正负之分。

矩形阵列

（1）快捷键：AR
（2）创建方式分为矩形阵列、路径阵列和环型阵列。
（3）阵列工具里，任何数值的改变必须用回车进行确认。
（4）注意关联选项，默认是开启的，阵列出来的图形是一个整体的。

路径及环形阵列

（1）快捷键：AR
（2）路径阵列不需要对象在路径上，注意路径阵列里分定距等分和定数等分。
（3）环形阵列可以更改旋转方向。
（4）环形阵列时默认对象随之发生旋转。

偏移工具

（1）快捷键：O
（2）与定距拾取功能类似，不同的是偏移是复制对象原本特性到指定的距离方向处。
（3）偏移对象是整体时会整个轮廓发生偏移。
（4）常用偏移工具复制对象到某一指定距离方向处

CAD精准绘图

正交模式

（1）快捷键：F8
（2）只能捕捉到水平或垂直的方向，是一种特殊的捕捉模式。
（3）在此模式下通常用来画水平或垂直直线，可以利用极坐标或者相对坐标画斜线。

极轴追踪

（1）快捷键：F10
（2）可以设置多种角度捕捉。既可以设置某角度倍数模式，也可以添加某个角度的捕捉
（3）一般绘图中是常开状态。

对象捕捉

（1）快捷键：F3
（2）帮助我们快速定位到精确的点。
（3）在对象捕捉图标上右键可以设置捕捉类型，除了最近点，其它捕捉类型一般都是常开状态。

对象捕捉追踪

（1）快捷键：F11
（2）对象捕捉就是指绘图时鼠标靠近该类型点会出现提示，比如端点，中点等
（3）对象追踪是指你把鼠标停在一个点上片刻，出现小十字符号时移开鼠标，每次鼠标经过该点的水平或垂直方向时会出现虚线提示，只是一次有效。

等轴侧捕捉

（1）用于绘制轴测图，轴测图指用二维线框表示三维模型。
（2）必须配合正交模式使用。
（3）切换各个面快捷键：F5。

CAD注释及标注

线性标注

（1）快捷键：DLI
（2）标注由尺寸线、尺寸界限、尺寸文字组成。
（3）标注两点之间的水平距离或者垂直距离，根据鼠标来进行判断。
（4）线性标注里的多行文字可以输入数字或者字母。
（5）可以用文字功能对标注文字更改，也可以直接双击标注文字进行编辑。注意，更改后不再显示真实值，应急处理不推荐。
（6）线性标注里的角度和旋转功能不一样，一个针对的是文字，一个针对的是尺寸界限。
（7）尺寸文字不允许有线穿过，可以采用倾斜样式。

对齐标注角度标注

（1）对齐标注快捷键：dal，标注两点之间的直线距离。
（2）角度标注快捷键：dan，标注两斜线之间的夹角。可以标注弧线的角度。

半径标注直径标注

（1）半径标注快捷键：dra
（2）直径标注快捷键：ddi。
（3）直径符号的文字输入：%%C。

标注样式

（1）标注样式管理器快捷键：D
（2）默认的标注样式不要更改，要新建一个属于自己的标注样式。
（3）可以根据需要自定义标注样式的各种属性包括箭头颜色等

引线标注

（1）快捷键：LE
（2）单引线标注在工具栏里找不到，只能快捷键启动。
（3）角度符号输入：%%d。
（4）引线标注里文字宽度为0，指的是采用文字的默认宽度。
（5）如果想要将引线文字调整到引线上方，则用分解工具X将引线分解后调整文字位置

全局比例

（1）自带的标注样式管理器里面的参数大多都不需要更改，都是国家标准。
（2）双击滚轮键（中键）可以显示整个图形。
（3）当文字或者箭头太小，应新建一个标注样式设置全局比例来调整整个标注样式。
（4）全局比例只能用于当前文档。
（5）全局比例只是调整显示的大小，而不修改其实际大小

单行及多行文字

（1）单行文字快捷键：DT
（2）单行文字可以设置旋转角度，每一行都是一个单独可以编辑的对象。
（3）多行文字快捷键：T
（4）多行文字创建的文字是一个整体，双击即可进入编辑。完成输入之后可以按ctrl+enter确定完成。

文字样式

（1）中文不支持大字体，其中开头带@的是垂直字体
（2）在文字样式下拉列表中打开文字样式管理器，跟标注样式类似，当前样式不允许删除，自带的标准文字样式不要更改，新建一个属于自己的文字样式。
（3）当遇到文字显示“？”的时候是因为字体库中不支持当前文字样式，可以更改文字样式里的字体。

CAD图层编辑

特性工具

（1）可以更改对象的颜色、线宽、线型
（2）没有的线型可以点击其他、点击加载。
（3）特性匹配快捷键：MA。类似格式刷，可以复制格式
（4）当线宽更改后要打开辅助状态栏里的线宽才能看到。

新建图层

（1）图层：类似于PS里的图层概念，一层一层透明的纸张叠加到一块的显示效果，这样可以把我们的图形按照层的概念进行分类。方便我们的单独进行编辑。
（2）没有的线型可以点击其他、点击加载。
（3）直接双击可以切换当前图层。
（4）单击图层特性按钮打开图层设置对话框，可以新建/删除/修改图层，修改其图层属性等

图层样板

（1）新建的图层是无法用于其他的文档。
（2）我们可以将设置好的图层信息导出成样板文件DWT。
（3）新建文件的时候直接选择我们设置好的样板文件进行新建即可。
（4）把设置好的样板文件保存到U盘里，形成一个良好的习惯，不用每次去设置绘图环境。

图层的运用

图层线型标准
粗实线：可见轮廓线
细实线：尺寸线，尺寸界限，剖面线，引出线
波浪线：断裂处的边界线，视图和剖视的分界线
虚线：不可见的轮廓线
点划线：轴线，对称中心线
双点划线：假想投影轮廓线，中断线。

图层操作

（1）开关冻结隔离锁定图层共有八种操作，是一一对应的关系。
（2）关闭图层主要为了保护该图层里面的对象，关闭之后该图层无法显示，无法操作
（3）冻结图层无法冻结当前图层。冻结之后该图层无法显示，无法操作
（4）锁定图层之后能画进去对象，但是无法删除或者编辑。
（5）隔离图层的作用是单独显示一个图层来编辑，其他图层全部隐藏锁定。

非连续线性比例

（1）快捷键：lts
（2）功能就是改变非连续线性的显示比例，调整虚线的显示间隔，针对的是全局的。
（3）CTRL+1快捷键打开对象特性工具栏查看对象所有属性，在特性工具栏里更改的是针对单独的某一个对象。
（4）可以用MA特性匹配工具进行多个对象的属性调整。

CAD块

块的认识

（1）在图纸中，很多时候同样的图形元素出现很多次，比如说机械图纸里面的螺栓、螺母，抑或是家装图里的家具、洁具等等，如果一个个去绘制的话效率就太低了，所以用到了块命令。
（2）可以理解块工具是批量复制，选择性粘贴。
（3）很多基于插件或者图库都是基于块工具开发的。

块的建立

（1）打开一张图档，确定要创建块的对象。
（2）单击“绘图”-“块”-“创建”命令，弹出块定义对话框。
（3）在块定义对话框中，输入“名称”，可自定义，单击“拾取点”，指定块的插入点。
（4）快捷键：B
（5）“按统一比例缩放”和“允许分解”按需要勾选

块的插入

（1）插入时候可以更改块的比例及角度。
（2）创建永久块：w。创建之后可以在所有CAD文件中插入块
（3）新建文件导入永久块需要单个选择，无法批量导入。

属性块

（1）简单的说，属性块就是在图块上附加一些文字属性（Attribute），这些文字可以非常方便地修改。。
（2）先建立可以更改定义的属性（快捷键ATTEDF），填写标记和属性默认值创建属性。
（3）将定义的属性和块放在一起共同创建块，就定义成了属性快，常用于插入注释符号
（4）插入属性快时，需要指定具体属性

动态块

（1）动态块的参数分为三块：参数、动作和约束。
（2）参数：包括点、线型、极轴、旋转、翻转、对齐、阵列，可见性、查询列表等。查询列表可以设置与图形的一个多个参数关联，对这些参数进行参数化控制。
（3）动作：移动、缩放、拉伸、阵列、翻转、查询等。
（4）约束：就是调用参数化相关功能，可以对块内图形添加垂直、相切、相等的约束条件，或者利用参数化控制图形尺寸。

CAD输出及打印

导出多种格式

（1）CAD可以导出多种格式，比如说dwg，dxf，dws，dwt。第一个较常用的标准文件格式。
（2）dwf格式，这种是选择打印机来保存的，CAD自带的，不过保存之后就不能修改了。
（3）同时可以导出图片格式，PDF格式的，这些一旦打印成，是不能直接修改的，一般用来打印成图时用。

模型空间打印

（1）显示：打印出视口显示图形。
（2）范围：打印画布中所有图形。
（3）窗口：选择打印的范围。
（4）打印快捷键：ctrl+p

布局打印

（1）可以新建多个布局，设置好之后可以批量打印。
（2）一个布局里可以新建很多视口进行补充表达。
（3）视口和视口之间是相互独立的，一次只能单独编辑一个视口。
（4）双击视口内部即可激活，退出视口双击空白的地方即可。
（5）小视口不好激活的时候可以移动出来，编辑后再放回去。
